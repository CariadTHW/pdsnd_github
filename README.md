### Description
This is a practice project to improve your Python skills.

This project uses Python to explore data related to bike-sharing systems in three major cities in the United States: Chicago, New York City, and Washington, D.C. You will write code to import the data and answer interesting questions by calculating descriptive statistics. You will also create a script that takes raw data as input to provide an interactive terminal experience and display these statistics.

### Files used
bikeshare_starter.py: file with basic python code. To be adapted during the project accordingly to the practice exercises.

### Dataset
Randomly selected data for the first six month of 2017 are provided for all three cities.
